# Спецификация ролей — ЗОВ Денег API

## Роли

| Роль        | Описание                                                                 |
|-------------|--------------------------------------------------------------------------|
| `anonymous` | Неаутентифицированный пользователь. Доступ только к публичным данным.   |
| `user`      | Аутентифицированный инвестор. Может торговать и управлять своим счётом. |
| `admin`     | Администратор системы. Полный доступ к управлению платформой.            |

## Бизнес-ограничения

### Роль `user`
- Видит **только свои** заявки, транзакции, портфель и баланс.
- Не может видеть данные других пользователей.
- Может отменить заявку только в статусе `pending`.
- Количество в заявке должно быть **кратно размеру лота**.
- При подаче заявки `buy`: баланс должен покрывать рыночную покупку и комиссию (по текущим котировкам на момент исполнения).
- При подаче заявки `sell`: в портфеле должно быть достаточно бумаг.

### Роль `admin`
- Не имеет торгового счёта — **не может торговать**.
- Не видит чужие портфели и балансы напрямую (только через будущий `/admin`-раздел v2).
- Не может заблокировать другого `admin` или самого себя.
- Не может удалить пользователя с открытыми позициями.
- Не может удалить бумагу, присутствующую в портфелях.

## Коды ошибок — справочник

| Код                       | HTTP  | Описание                                              |
|---------------------------|-------|-------------------------------------------------------|
| `VALIDATION_ERROR`        | 400   | Ошибка валидации входных данных                       |
| `PIN_MISMATCH`            | 400   | Новый PIN и подтверждение не совпадают                |
| `PIN_SAME_AS_CURRENT`     | 400   | Новый PIN совпадает с текущим                         |
| `INVALID_LOT_SIZE`        | 400   | Количество не кратно размеру лота                     |
| `INSUFFICIENT_FUNDS`      | 400   | Недостаточно средств на счёте                         |
| `INSUFFICIENT_SECURITIES` | 400   | Недостаточно бумаг в портфеле для продажи             |
| `INVALID_AMOUNT`          | 400   | Некорректная сумма операции                           |
| `INVALID_DATE_RANGE`      | 400   | Некорректный интервал: `from` > `to` (Unix timestamp, сек), в т.ч. для истории цен |
| `ORDER_NOT_CANCELLABLE`   | 400   | Заявку в текущем статусе нельзя отменить              |
| `INVALID_CURRENT_PIN`     | 401   | Неверный текущий PIN-код при смене                    |
| `INVALID_PIN`             | 401   | Неверный PIN-код при входе                            |
| `UNAUTHORIZED`            | 401   | Отсутствует или просрочен access token                |
| `REFRESH_TOKEN_EXPIRED`   | 401   | Refresh token истёк                                   |
| `INVALID_REFRESH_TOKEN`   | 401   | Refresh token недействителен                          |
| `FORBIDDEN`               | 403   | Недостаточно прав                                     |
| `CANNOT_BLOCK_ADMIN`      | 403   | Нельзя заблокировать администратора                   |
| `USER_NOT_FOUND`          | 404   | Пользователь не найден                                |
| `SECURITY_NOT_FOUND`      | 404   | Ценная бумага не найдена                              |
| `NOT_FOUND`               | 404   | Ресурс не найден                                      |
| `EMAIL_ALREADY_EXISTS`    | 409   | Email уже зарегистрирован                             |
| `PHONE_ALREADY_EXISTS`    | 409   | Телефон уже зарегистрирован                           |
| `ALREADY_IN_WATCHLIST`    | 409   | Бумага уже в списке отслеживания                      |
| `TICKER_ALREADY_EXISTS`   | 409   | Бумага с таким тикером уже существует                 |
| `MARKET_CLOSED`           | 422   | Рынок закрыт, торги недоступны                        |
| `TRADING_SUSPENDED`       | 422   | Торги по бумаге приостановлены                        |
| `ACCOUNT_LOCKED`          | 423   | Аккаунт временно заблокирован (слишком много попыток) |
| `INTERNAL_ERROR`          | 500   | Внутренняя ошибка сервера                             |
